MySQL8限定IP访问

发表于2020-05-03|更新于2020-11-02|MySQL
|字数总计:132|阅读时长:1分钟|阅读量:
  • MySQL版本：8.0.13

  1. 登录Mysql。

  2. 切换database。

  3. 创建用户同时指定可访问的ip,可以用%来通配某段域。

  4. 给刚创建的 user@ip 组合赋予所有权限。

  5. 刷新生效。

  6. 查看用户表。

用户可以重复创建，user@ip 组合唯一即可。
