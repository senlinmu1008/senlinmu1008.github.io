卸载MySQL8

发表于2020-04-08|更新于2020-11-02|MySQL
|字数总计:95|阅读时长:1分钟|阅读量:
  • Linux版本：CentOS7.6
  • MySQL版本：8.0.13

1、卸载安装包

  1. rpm -qa|grep mysql查看mysql安装包。

  2. 卸载。

2、删除相关文件夹

  1. find / -name mysql查询所有mysql相关文件。

  2. rm -rf /var/lib/mysql即可。
