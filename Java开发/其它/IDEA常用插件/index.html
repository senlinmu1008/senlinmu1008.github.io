IDEA常用插件

发表于2020-07-29|更新于2020-11-02|Java开发
|字数总计:242|阅读时长:1分钟|阅读量:

列举IDEA中常用的开发插件。可以在IDEA中自行选择下载安装。

1、Statistic

用于统计工程代码行数。总行数、注释行数、空行数都会统计列举出来。

2、Maven Helper

可以更直观地查看项目中的所有Maven依赖，帮助我们更好地排除依赖的冲突。
支持列表和树形结构展示依赖关系，同时支持搜索功能、一键排除冲突依赖。

3、Lombok

编译器插件，通过一些注解比如@Data来修改字节码文件，自动生成get/set方法，使源码更加地简洁。

4、Alibaba Cloud Toolkit

阿里云自研的部署插件。
支持一键完成编译、打包、FTP上传服务器指定目录、执行启动命令、查看日志等操作。
同时还支持SSH、FTP等功能。

5、CamelCase

各种规则的变量命名快捷转换。快捷键 Alt+Shift+U，也可以自定义。
